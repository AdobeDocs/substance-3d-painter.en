Multi/Bi-GPU

Some GPU configurations and/or GPU models are incompatible with Substance 3D Painter and will lead to instabilities and crashes. Below is a list of the incompatible configurations :

Configuration Solution

Nvidia SLI / AMD Crossfire
(Graphic card bridges)

Disable SLI or Crossfire in the GPU driver settings.
Bi-GPU
(Two GPU chipsets on one graphic card) 		Disable the use of the two GPU chipsets in the drivers settings to only one.
