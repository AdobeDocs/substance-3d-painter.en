# Installing the particle editor

To create and edit particle system behavior, the PopcornFX editor need to be used.  
Download the editor from the one of the following links: [Mirror 1](http://downloads.popcornfx.com/updates/1.5.5.20787/PopcornFx-Editor_Setup_v1.5.5.20787.exe), [Mirror 2](https://www.dropbox.com/s/z1vwzvk2ywlug4p/PopcornFx-Editor_Setup_v1.5.5.20787.exe?dl=1).

>[!NOTE]
>
> Painter use an old version of the editor because it needs the support of a specific version. For more information see as well the [official wiki](https://wiki.popcornfx.com/).
