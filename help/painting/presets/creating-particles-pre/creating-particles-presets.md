# Creating particles presets

The particle system that allows to paint particles uses  **PopcornFX** which relies on **.pkfx** files to define the behaviors.

The particles brushes are composed of two parts: the **Emitter** and the **Receiver** which are saved together inside the preset. The application loads the Emitter and overrides collision events to emit the Receiver when painting.
