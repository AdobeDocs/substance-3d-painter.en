# Advanced channel painting

A few of the default channels used in shaders can be painted on to created advanced or complex effects. For example painting height information that get converted into a normal map.

The current channels available:

* [Ambient Occlusion Painting](../../help/painting/advanced-channel-painting/ambient-occlusion-pai/ambient-occlusion-painting.md)
* [Flow Map Painting](../../help/painting/advanced-channel-painting/flow-map-painting/flow-map-painting.md)
* [Height Map Painting](../../help/painting/advanced-channel-painting/height-map-painting/height-map-painting.md)
* [Normal Map Painting](../../help/painting/advanced-channel-painting/normal-map-painting/normal-map-painting.md)
