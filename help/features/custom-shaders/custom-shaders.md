# Custom Shaders

![](header-shader.jpg)

Substance Painter uses shaders to render materials in its realtime viewport.

It is possible to write custom shaders to implement new behaviors or to simply make the viewport match other renderers. Additional shaders for Substance Painter can be found on [Substance Share](https://share.allegorithmic.com/libraries?by_category_type_id=6).

## Creating new Shaders

Creating new custom shaders is possible by simply creating new **.glsl** files.

A detailed [shader API](https://helpx.adobe.com/substance-3d/unlisted/documentation/spdoc/custom-shader-api-89686018.html) is available and provide helper functions to create new effects and integrate into the existing workflow.
