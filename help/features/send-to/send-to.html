Send to

Send to are actions to easily export and import resources from a project into another application, avoiding the need to do it manually. Send to can be used to export the 3D model and its textures into another rendering application, or to recieve Substance materials from Substance 3D Designer.

Send to actions are available via the File > Send to menu.

The currently supported Send to are:

  • Send to Photoshop
  • Send to Substance 3D Stager
  • Send to After Effects 

Painter can also recieve resources from other applications, such as:

  • Substance 3D Designer
  • Substance 3D Sampler

For more information on recieving resources, see the dedicated documentation page.

Note

Some of the Send to workflows are only available with the Adobe version of Painter because of technical dependencies with the Creative Cloud Desktop software.

Also some actions in the Send to menu may not be available if the required software is not installed.
