# Creating custom effects

Substance effects must be built in a specific way in  **Substance 3D Designer**  to work correctly once imported in Substance 3D Painter.

>[!NOTE]
>
> Custom effects can be imported in a project of the default shelf, see : [Importing assets](https://helpx.adobe.com/substance-3d/unlisted/documentation/spdoc/adding-content-to-the-shelf-142213317.html)
