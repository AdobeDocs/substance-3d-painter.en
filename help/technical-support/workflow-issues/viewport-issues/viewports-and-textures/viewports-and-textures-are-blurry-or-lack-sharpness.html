Viewports and textures are blurry or lack sharpness

The viewports may appear blurry for different reasons.

High-DPI screens (Retina) settings

By default Substance 3D Painter downscale the viewport resolution on High-DPI/Retina screen to improve performances.

This behavior can be changed in the main settings by changing the Viewport Scaling parameter.

Texture filtering

The viewports use mipmaps and texture filtering to be able to stream in and out Sparse Virtual Textures to improve performances. This can lead to blurry textures in some cases.

The texture filtering can be adjusted via the Display Settings window under the Viewport Settings parameters.
