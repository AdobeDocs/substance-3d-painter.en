# Exporting a DXDiag

This page explains what a DXDiag is and how to generate it.

The DXDiag is a diagnostic created on Windows that gives information about your computer and your GPU setup.

Here are the steps to export it when requested by the support team:

1. Go to Start &gt; Run and type **"dxdiag"** (without quotes) in the **"Open"** field and press OK button
1. Wait for the **"green"** progress bar to finish loading and then press the **"Save All Information..."** button
1. Name the file with some useful info, like your name and the date, and save it where you can find it, like the Desktop, by pressing the **"Save"** button
1. Press the **"Exit"** button to quit this application
