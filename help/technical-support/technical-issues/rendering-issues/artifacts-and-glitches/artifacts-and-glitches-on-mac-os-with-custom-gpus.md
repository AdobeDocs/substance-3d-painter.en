# Artifacts and glitches on Mac OS with Custom GPUs

>[!WARNING]
>
> Non-official Apple configurations are not officially supported, meanings any part (for example the GPU) that has not been sold originally by Apple may not work properly with the software.

In case glitches, crashes or even artifacts appear on Mac OS when running a custom GPU with custom drivers a workaround may be to delete the **properly list** files related to the application.

To know which file to remove, see: [Preferences and application data location](https://helpx.adobe.com/substance-3d/unlisted/documentation/spdoc/application-preferences-location-147095594.html).
