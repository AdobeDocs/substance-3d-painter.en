Add resource paths by editing preferences manually

It is possible to add additional custom Shelf or Assets paths without opening the application.

To do so, it requires to edit the application preferences (which are stored differently per platform). Once added, the new shelf should appear in the shelf list in the main preferences. See this page for more details : Libraries Configuration

Note

Network path are not supported, be sure to use a mounted path to avoid any issues. See : Loading files from a network.
