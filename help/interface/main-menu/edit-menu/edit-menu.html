Edit menu


The edit menu allows to access quickly the undo/redo actions but also to access the project settings and the global settings.

ActionDescription
UndoGo one step back in the History stack.
RedoGo one step forward in the History stack.
Project configurationOpen the project settings window of the current project.
SettingsOpen the general application settings window.
Bake Mesh mapsOpen the Baking window.
