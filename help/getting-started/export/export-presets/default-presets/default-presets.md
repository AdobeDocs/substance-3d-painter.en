# Default Output templates

Below is a list of additional Output templates provided by default.  Default Output templates can be copied and modified in the <b>Output templates tab </b>of the <b>Export window</b>.

>[!NOTE]
>
> Some other software not listed here may also be supported (such as  **Octane**). For more information see the [ecosystem and plug-ins](https://helpx.adobe.com/substance-3d-integrations/home.html).

| *Preset* | *Description* |
| --- | --- |
| <b> Amazon Lumberyard </b> | Name and pack textures to be imported and used in Amazon Lumberyard. |
| <b> Arnold (AiStandard) </b> | Preset for Arnold (AiStandard shader). |
| <b> Arnold UDIM Legacy (AiStandard)</b> | Preset for Arnold UDIM Legacy (AiStandard). |
| <b>Blender (Principled BSDF)</b> | Preset for Blender (Principled BSDF). |
| <b> Corona </b> | Preset for Corona renderer. |
| <b> CryEngine 3 </b> | Name and pack textures to be imported and used in Cry Engine 3. |
| <b> Dota 2 </b> | Preset for the game Dota 2 (can be used with the associated template). |
| <b> Keyshot </b> | Preset for Keyshot renderer. |
| <b>Keyshot 9+</b> | Preset for Keyshot renderer from version 9 onwards. |
| <b> Lens Studio </b> | Preset for Lens Studio (to create assets for Snapchat). |
| <b>Maxwell (Metallic Roughness)</b> | Preset for Maxwell renderer using the Metallic Roughness workflow. |
| <b>Maxwell (Specular Glossiness)</b> | Preset for Maxwell renderer using the Specular Glossiness workflow. |
| <b> Mesh Maps </b> | Export the baked mesh map slots of a texture set individually (exported at the same resolution as for the texture set). |
| <b>Non-PBR Specular Glossiness</b> | Name and export textures for a non-PBR Specular Glossiness workflow. |
| <b> PBR Metallic Roughness</b> | Name and export textures needed for a regular Metallic/Roughness PBR workflow. |
| <b> PBR Metallic Roughness (Folder or PSD Groups) </b> | Similar to PBR MetalRough, but creates an additional folder. This preset is mostly suited to be used with the PSD file format. |
| <b> PBR Specular Glossiness</b> | Name and export textures needed for a regular Specular/Glossiness PBR workflow. (The textures are converted from the BaseColor/Metallic workflow). |
| <b> PBR Specular Glossiness (converted from Metallic Roughness</b> | Convert Basecolor, Metallic and Roughness channels to be compatible with the Specular/Glissiness workflow. |
| <b> Redshift </b> | Preset for Redshift renderer. |
| <b>Renderman (pxrDisney)</b> | Preset for Renderman using Metallic and Rougness maps. |
| <b>Renderman (pxrSurface)</b> | Preset for Renderman using Diffuse and Specular maps. |
| <b>Roblox (MaterialVariant)</b> | Preset for Roblox using Albedo without alpha support. |
| <b>Roblox (SurfaceAppearance)</b> | Preset for Roblox using Colormap with alpha channel. |
| <b> Shade 3D </b> | Preset for Shade 3D. |
| <b>Spark AR Studio</b> | Preset for Spark AR Studio. |
| <b>Unity HD Render Pipeline (Metallic Standard)</b> | Preset for Unity's HD render pipeline using Metallic and Glossiness maps. |
| <b>Unity HD Render Pipeline (Specular)</b> | Preset for Unity's HD render pipeline with added specular map. |
| <b>Unity Universal Render Pipeline (Metallic Standard)</b> | Preset for Unity's Universal render pipeline using Metallic and Smoothness maps. |
| <b>Unity Universal Render Pipeline (Specular)</b> | Preset for Unity's Universal render pipeline with Specular and Smoothness maps. |
| <b> Unreal Engine (Packed) </b> | Name and pack textures to be used in the Unreal Engine with the lowest possible memory footprint. |
| <b>Unreal Engine SSS (Packed)</b> | Name and pack textures to be used in the Unreal Engine with the lowest possible memory footprint, replacing opacity map with subsurface scattering. |
| <b> Vray Next (Metallic Roughness)</b> | Preset for Vray renderer using PBR workflow. |
| <b>Vray Next (Specular Glossiness)</b> | Preset for Vray renderer using non-PBR workflow. |
| <b> Vray UDIM Legacy (Metallic Roughness)</b> | Preset for Vray renderer (UDIM naming convention). |
