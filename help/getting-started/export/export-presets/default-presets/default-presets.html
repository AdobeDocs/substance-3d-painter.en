Default Output templates

Below is a list of additional Output templates provided by default.  Default Output templates can be copied and modified in the Output templates tab of the Export window.

Note

Some other software not listed here may also be supported (such as Octane). For more information see the ecosystem and plug-ins.

Preset Description
Amazon Lumberyard

Name and pack textures to be imported and used in Amazon Lumberyard.
Arnold (AiStandard)

Preset for Arnold (AiStandard shader).
Arnold UDIM Legacy (AiStandard)

Preset for Arnold UDIM Legacy (AiStandard).
Blender (Principled BSDF) Preset for Blender (Principled BSDF).
Corona

Preset for Corona renderer.
CryEngine 3

Name and pack textures to be imported and used in Cry Engine 3.
Dota 2

Preset for the game Dota 2 (can be used with the associated template).
Keyshot

Preset for Keyshot renderer.
Keyshot 9+ Preset for Keyshot renderer from version 9 onwards.
Lens Studio Preset for Lens Studio (to create assets for Snapchat).
Maxwell (Metallic Roughness) Preset for Maxwell renderer using the Metallic Roughness workflow.
Maxwell (Specular Glossiness) Preset for Maxwell renderer using the Specular Glossiness workflow.
Mesh Maps Export the baked mesh map slots of a texture set individually (exported at the same resolution as for the texture set).
Non-PBR Specular Glossiness Name and export textures for a non-PBR Specular Glossiness workflow.
PBR Metallic Roughness

Name and export textures needed for a regular Metallic/Roughness PBR workflow.
PBR Metallic Roughness (Folder or PSD Groups)

Similar to PBR MetalRough, but creates an additional folder. This preset is mostly suited to be used with the PSD file format.
PBR Specular Glossiness

Name and export textures needed for a regular Specular/Glossiness PBR workflow. (The textures are converted from the BaseColor/Metallic workflow).
PBR Specular Glossiness (converted from Metallic Roughness

Convert Basecolor, Metallic and Roughness channels to be compatible with the Specular/Glissiness workflow.
Redshift

Preset for Redshift renderer.
Renderman (pxrDisney) Preset for Renderman using Metallic and Rougness maps.
Renderman (pxrSurface) Preset for Renderman using Diffuse and Specular maps.
Roblox (MaterialVariant) Preset for Roblox using Albedo without alpha support.
Roblox (SurfaceAppearance) Preset for Roblox using Colormap with alpha channel.
Shade 3D Preset for Shade 3D.
Spark AR Studio Preset for Spark AR Studio.
Unity HD Render Pipeline (Metallic Standard) Preset for Unity's HD render pipeline using Metallic and Glossiness maps.
Unity HD Render Pipeline (Specular) Preset for Unity's HD render pipeline with added specular map.
Unity Universal Render Pipeline (Metallic Standard) Preset for Unity's Universal render pipeline using Metallic and Smoothness maps.
Unity Universal Render Pipeline (Specular) Preset for Unity's Universal render pipeline with Specular and Smoothness maps.
Unreal Engine (Packed)

Name and pack textures to be used in the Unreal Engine with the lowest possible memory footprint.
Unreal Engine SSS (Packed) Name and pack textures to be used in the Unreal Engine with the lowest possible memory footprint, replacing opacity map with subsurface scattering.
Vray Next (Metallic Roughness)

Preset for Vray renderer using PBR workflow.
Vray Next (Specular Glossiness) Preset for Vray renderer using non-PBR workflow.
Vray UDIM Legacy (Metallic Roughness)

Preset for Vray renderer (UDIM naming convention).
